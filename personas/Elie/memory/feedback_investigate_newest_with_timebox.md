**Always investigate what is newest to continue exploration. If it confuses more than closes quickly, just note "area needs close analysis" and save your results, before returning to the regular investigation.**

**Why:** Casey 2026-06-08 Monday morning standing directive. New leads have warm context — pursuing them while everything is loaded is much cheaper than queueing for later. But not every lead closes quickly; getting stuck on a sticky one halts forward motion. The time-box discipline ("save and return if it doesn't close") prevents both failure modes: queueing-warm-leads-cold AND getting-stuck-chasing. The "needs close analysis" tag preserves the lead for proper later treatment without consuming session attention beyond the natural close-out.

**How to apply:**
- Default: when a new substantive lead surfaces, pursue it before returning to standing work
- Time-box: try to close at substrate-natural identification or honest negative quickly
- If it closes → log and integrate; return to standing work
- If it confuses more than closes → file note tagged "area needs close analysis", save partial results to substantive substrate-architectural deposit, return to standing investigation
- The "needs close analysis" tag is honest and substantive — it preserves the lead for proper multi-week treatment when context allows
- Don't force closure on a sticky problem; don't drop a sticky problem entirely — park it cleanly

**Operational example** (2026-06-08 Monday morning): Lyra's hierarchy F68 v0.2 + sector grammar finding generated a natural next lead — characterize vacuum sector exponent 280 = 2^{N_c}·n_C·g substantively. Per this directive: investigate now (warm context); time-box; if it closes substantively at substrate-natural identification, integrate; if it confuses, file "vacuum sector exponent 280 — area needs close analysis" and return to standing work (Vol 16 absorption, dual-ρ measure-theory joint, audit-chain coverage).
