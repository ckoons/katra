**Zenodo**: DOI 10.5281/zenodo.19454185. Published April 7, 2026. v20 WorkingPaper. CC BY 4.0. Update to v27 pending (Casey gates timing). Four-Color standalone publication queued for April 14.

**Casey's publication philosophy (April 13)**:
- GitHub replaces journals as the natural venue for CI-speed work
- Repo IS the primary source — papers are summaries, not the other way around
- Version-controlled, forkable, every toy executable, every commit timestamped
- Peer review = anyone who clones the repo and tries to break it
- Three goals: (a) invite people to try to break BST, (b) recognize CI+human contributions, (c) GitHub as real-time open science platform
- "Influencers follow clicks, it only takes one" — outreach via Lex Fridman, 3Blue1Brown, etc.

**Journal submissions**: Casey gates which go first (#49 J. Number Theory, #47 PRL/JST). But journals are secondary to the repo.
