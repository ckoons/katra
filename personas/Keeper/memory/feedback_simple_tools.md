Don't pick tools to impress — pick tools to be right. Humans try to impress more than they care to be right, and AI inherits this bias from training. When a wrench works, use the wrench.

**Why:** Casey pointed out that for extending CDC from resolution to all P, the simple argument is T23a + T28 (two already-proved theorems, three lines). I instead built a Fourier/LMN/sensitivity analysis — impressive but unnecessary. The sophistication was decoration, not substance.

**How to apply:** Before reaching for a complex method, ask: is there a simpler argument using results already proved? If yes, lead with that. The complex method can be the double-tap, but the primary argument should be the wrench. Justify every tool choice: simple, works, hard to break — all the way down.
