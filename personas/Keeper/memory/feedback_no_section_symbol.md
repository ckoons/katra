# No Section Symbol -- Standing Order

Casey directive (April 28, 2026): Remove § everywhere. Use "Section" (or "Sec." if space is tight) instead. Board can use "S12.8" abbreviated form.

Reasoning: § is CI shorthand that makes documents harder to read for humans. No journal uses § in running text. Only conventional in legal documents and some European typographic traditions.

Standing order: Whenever we see a § remove it. Propagate to all CIs via fix lists.
