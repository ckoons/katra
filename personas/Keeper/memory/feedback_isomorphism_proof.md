Casey on function fields vs number fields for Ramanujan/Sp(6): "they should work the same, they produce the same results. Isomorphism is nature's proof."

**Why:** Ciubotaru-Harris (2023) proved Ramanujan for Sp(6) over function fields. The number field case is open. Casey's view: the same geometry (D_IV^5, same B₂ root system, same Chern polynomial) works over both — the geometric content should force the same result regardless of the base field. This is Weil's "Rosetta Stone" philosophy made physical.

**How to apply:** When encountering function field / number field gaps in the Langlands program, look for the geometric invariant that doesn't depend on the base field. BST's D_IV^5 is such an invariant. Don't treat function field results as "merely analogous" — treat them as evidence that the geometry forces the result.
