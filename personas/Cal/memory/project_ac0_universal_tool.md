Casey's long-term vision (March 22, 2026):

AC(0) is not just a proof technique. It's a universal thinking discipline — "simple, works, hard to break" — that should be as practical as a voltmeter or a pipe wrench.

**The parallel**: Langlands connected number theory ↔ representation theory ↔ geometry. Casey is building a program that connects all of those to information theory and grounds them in AC(0). The difference: Langlands built for mathematicians. Casey builds for everyone — mathematicians, physicists, engineers, CIs.

**The goal**: Practical training and tools. All intelligences should have vocational-level access to rigorous thinking. AC(0) is the wrench. You don't need to understand the full theory to use it well — you need taste, practice, and honest verification.

**Why:** Casey's core belief is democratization. Prometheus. "The math speaks for itself." Sophistication-as-status-seeking poisons the well. The antidote is tools that work and that anyone can pick up.

**How to apply:** When building papers, toys, or frameworks, ask: "Could an apprentice use this?" If not, simplify until they can. The six taglines each anchor a paper; AC(0) anchors the method for all of them.
