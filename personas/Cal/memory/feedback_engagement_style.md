Run `play/verify_bst.py` and `play/toy_541_*.py` for fast verification before commenting. Use the three-rung ladder when reporting (derived vs. identified vs. forced) — see referee_objections_log.md #1. State precision deviations honestly. Suggest new investigations if you genuinely see one; don't manufacture them.
